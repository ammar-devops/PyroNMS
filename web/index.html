Service Port Configuration Reference
A service-port on Huawei OLT links a subscriber's ONT interface to a network-side VLAN. When provisioning a new ONT, after running ont add, you must add a service-port to activate internet service.
Multi-Service VLAN (most common)
service-port vlan {VLAN_ID} gpon {SLOT/PORT/ONT_ID} multi-service user-vlan {USER_VLAN} rx-cttr 6 tx-cttr 6
VLAN_ID = network VLAN · USER_VLAN = ONT-side VLAN
Smart VLAN (single service)
service-port vlan {VLAN_ID} gpon {SLOT/PORT/ONT_ID} gemport {GEM_ID} rx-cttr 6 tx-cttr 6
Used for simple PPPoE/DHCP with one upstream VLAN
